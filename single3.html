Python Web Application Creation and Deployment

Project Overview: The goal of this project is to create a Python web application that translates text between English and French using the deep_translator Python package. The project involves creating server-side functionality, writing unit tests, and deploying the application in a lab environment.

GitHub Link to the project: Python application

Key Steps and Components:
1. Translator Implementation:
• Imported and created an instance of the MyMemoryTranslator in the translator.py file.
• Implemented two functions in translator.py: One to translate from English to French, another to translate from French to English.
2. Unit Tests:
• Wrote unit tests to ensure the accuracy of the translation functions.
3. Coding Standards Check:
• Ran a coding standards check against the implemented functions using pylint.
4. Package Creation:
• Packaged the translation functions and tests
5. Server Implementation:
• Imported the created package in server.py.
• Created two endpoints (/englishToFrench and /frenchToEnglish) in server.py to handle translation requests.
6. Local Testing:
• Tested the endpoints on the local server to ensure that the translation functionality works as expected.
7. Lab Environment Deployment:
• Deployed and tested the application in a lab environment to ensure its functionality in a production-like setting.
